Vivaldi

Vivaldi Browser is a freeware browser developed by Vivaldi Technologies. The project was started by the co-founder of Opera Software (Jon Stephenson von Tetzchner), due to disagreements with the direction that the Opera Browser software project was taking. The browser is aimed at power users, and former users of Opera. Vivaldi offers extensive customizabilty of the user interface.
